# HireSense

HireSense is an **AI-assisted hiring intelligence platform** designed to improve recruitment efficiency through smarter screening and structured evaluation.

---

## 🧩 Modules

* **TalentIQ (Screening)**
  Resume analysis, scoring, and candidate ranking with alternative role suggestions.

* **CandidateIQ (Evaluation)**
  AI chatbot-based interviews with response evaluation and report generation.

> ⚠️ Modules currently operate independently (integration planned).

---

## 🔁 Workflow

JD Upload → Resume Screening → Shortlisting → AI Interview → Report → Decision

---

## 🏗️ Architecture

Frontend (HR Dashboard, Candidate UI)
→ Backend (Processing Engines)
→ AI Layer (GPT-4o-mini)
→ Database (Supabase)

---

## ⚙️ Tech Stack

* React
* Node.js / Python
* GPT-4o-mini
* Supabase

## 🧠 AI Usage
